# Acoustigrain
An IoT-based system built for rice weevil detection in rice-grain storage

In fulfilment of the capstone project development:
AcoustiGrain: A Non-Invasive Internet of Things (IoT) Bio-Acoustic Signature Detection System for Rice Weevil (Sitophilus oryzae) Infestation Using Fast Fourier Transform (FFT)

The primary goal of this study is to develop a non-invasive acoustic monitoring system for early pest detection in rice storage. Specifically, the project aims to achieve the following objectives:
To design and develop a tapered wedge-form factor hardware capable of being positioned between 50kg rice sacks without compromising the physical integrity of the woven polypropylene fibers or breaking official seals.
To implement a Digital Signal Processing logic using the Fast Fourier Transform (FFT) Algorithm on an ESP32-S3 microcontroller to isolate the acoustic signatures of Sitophilus oryzae (Bukbok) from ambient warehouse noise.
To establish a long-range wireless sensor network using LoRaWAN technology for the real-time transmission of acoustic data from multiple storage nodes to a centralized 3D Heat Map Dashboard.
To evaluate the technical performance and software quality of the AcoustiGrain system based on the ISO/IEC 25010 standard, focusing on functional suitability, reliability, performance efficiency, and usability.
